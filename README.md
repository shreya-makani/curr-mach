# Currency Converter Machine

This machine converts an amount from one currency to another using live exchange rates.

## Features
- Converts currency using real-time exchange rates.
- Supports multiple currencies.

## Usage
1. Set the `amount`, `from_currency`, and `to_currency` in `input.json`.
2. Run the machine.
3. Check `output.json` for the converted amount and exchange rate.

## Requirements
- Python 3.10
- Internet connection for fetching exchange rates.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the machine: `python main.py`
